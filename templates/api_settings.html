Настройки API для разработчиков

Документация API

Для интеграции вашей сборки с маркетплейсом используйте следующие эндпоинты:

1. Вход в систему

POST /api/v1/login
Body: {"username": "your_username", "password": "your_password"}
Response: {"token": "jwt_token", "user": {...}, "builds": [...]}

2. Получение списка сборок

GET /api/v1/builds
Headers: Authorization: Bearer YOUR_JWT_TOKEN

3. Покупка сборки

POST /api/v1/purchase/{build_id}
Headers: Authorization: Bearer YOUR_JWT_TOKEN

Информация для лаунчера

Ваш лаунчер должен:

  1. Отправить логин/пароль на /api/v1/login
  2. Получить JWT токен из ответа
  3. Использовать токен для всех последующих запросов
  4. Получить список доступных сборок через /api/v1/builds
